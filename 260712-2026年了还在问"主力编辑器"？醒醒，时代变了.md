「作为一名开发者，你目前的主力代码编辑器是哪个，为什么选择它？」

但现在？这个问题本身，就是最大的问题。

# 你还在选编辑器，有些人已经不写代码了

2026年了，如果一个开发者日常工作的核心动作还是"打开编辑器、敲代码"，那他大概率还在用2020年的方式上班，只是电脑换新了而已。

我自己的开发日常是这样的：

打开终端，对着 antigravity cli 说一句"把这个模块的认证逻辑重构一下，用OAuth 2.1"。

然后我去倒杯水。

回来的时候，代码改完了，测试跑完了，diff 整整齐齐地等着我看。

我要做的，只是审查;不是"编辑"。是审查。

代码不需要被"编辑"了，它只需要被"查阅"。

这是我从今年二月份开始就一直在实践的工作流。VS Code？卸了。IDE？更不可能。Cursor？没碰过。

# 2026年真正该问的问题

知乎那个问题，如果让我来重新出题，我会问一串完全不同的东西：

第一，你的主力 Coding Agent 是哪个？

Claude Code？Codex？Antigravity？Open Code？还是别的什么？

这才是2026年"编辑器之争"的真正战场。只不过，被替换掉的不是一个软件，而是整个工作范式。

我个人是多 Agent 轮番上阵的打法。复杂的架构设计和代码重构，我扔给 坐拥Fable 5的Claude Code；其它？antigravity cli + gemini 是我的首选。

就像当年你不会只用一把螺丝刀，现在你也不该只绑定一个 Agent。

第二，你的编程模型选的哪个？

Opus？Gemini？GPT？GLM？Kimi？还是开源的Qwen？

这个选择，比"VS Code装什么插件"重要一万倍。

因为模型决定了你的 Agent 的"智商上限"。同样一个重构任务，丢给不同的模型，出来的代码质量天差地别。

我个人目前最依赖的是 Claude 的 Opus 系列和 Google 的 Gemini。但模型迭代太快了，上个月的最优解，这个月可能就被刷下来了。保持灵活，别站队。

第三，你的语音输入法用的哪个？

这个问题可能会让很多程序员觉得莫名其妙。语音输入？写代码？

没错。

我现在跟 Agent 沟通，大量使用语音输入。你想想，你跟一个"下属"布置任务的时候，是打字快还是说话快？

当编程变成了"跟AI对话"，输入效率的瓶颈就从"打字速度"变成了"说话速度"和"表达清晰度"。

我日常用的是微信输入法。

第四，你的麦克风用的哪款？

既然语音成了主要输入方式，麦克风的质量就直接影响你的"编程效率"了。

这话说出来很魔幻，但它就是2026年的现实。

十年前的程序员讨论机械键盘的轴体手感，2026年的程序员应该讨论麦克风的拾音清晰度。时代变了，"人机接口"也变了。

# 那编辑器就彻底没用了吗？

当然不是。

代码虽然不需要被编辑了，但它依然需要被查阅。

你得看 Agent 改了什么，你得 Review diff，你得在脑子里把整个系统的脉络理清楚。这时候你需要一个速度快、界面干净、打开大文件不卡的工具。

而用来看代码，我选择的是Sublime Text 4。

为什么？因为它够快、够漂亮、够干净。

启动秒开，搜索秒出，十万行的文件丝滑滚动，没有任何多余的弹窗、提示、AI小助手在旁边嗡嗡叫。

Sublime Text 是我见过的最优雅的"只读工具"。它不试图帮我写代码，不试图帮我补全，不试图在每一行后面插一个灰色的建议——它就安安静静地把代码呈现在那里，让我专注地阅读和思考。

这种克制，在2026年反而成了一种稀缺的美德。

回想一下，当你坐到技术管理者的位置时，你不会亲自去写 if (x == y)。你的工作是审查、决策、把控方向。

过去，接你指令的是对面工位的工程师。

现在，接指令的是 Claude 和 Gemini。

区别在于：这些"下属"不需要午休，不会请假，永远不抱怨加班，而且你可以同时指挥好几个。

哪怕你只是一个独立开发者，从来没管过人，AI 时代也逼着你学会"管理"。管理 Agent，管理上下文，管理任务拆解，管理代码质量的验收标准。

每一个用 Agent 写代码的人，本质上都是一个"技术经理"。

只不过你管的不是人，是AI。

# 不要沉迷于"更快地搬砖"

为什么我始终不碰 Cursor 这类工具？

因为它们的核心逻辑还是"帮你更快地写代码"。更智能的补全、更丝滑的内联编辑、更方便的重构……

格局小了。

这就好比，马车时代，有人发明了一种人体工学马鞭，握感极佳，挥动省力，每一鞭都能让马多跑两步。

很好。但汽车已经发明了。

Cursor 是那根完美的马鞭。Agent 是汽车。

你选哪个？

# 写在最后

2026年的开发者工具链之争，早就不是编辑器之争了。它是一场关于工作范式的革命。

所以下次有人问你"主力编辑器是什么"，你可以像我一样回答：

我的主力编辑器是 Sublime Text 4——因为我只需要看代码。

至于写代码这件事？

交给 Agent 了。我只负责指点江山。

你觉得呢？

我是玩家翁伟。
