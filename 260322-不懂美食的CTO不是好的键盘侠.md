# 不懂美食的CTO不是好的键盘侠

_——从管理技术团队到管理一群AI，一个中年程序员的二十年与这一年_

我的在知乎上的自我介绍一直是这句话：**不懂美食的CTO不是好的键盘侠。**

听起来有点割裂，但这大概就是我最真实的状态。主业干了十几二十年技术，在几家公司担任过CTO，其中包括跨境电商和A股上市企业；业余因为太喜欢吃，十几年前就开始做自媒体，在网上写美食；后来兴趣又飘到了时政经济上，跑去知乎跟人指点江山。本来知乎给我认证的是“美食话题优秀答主”，结果后台数据默默把我归到了时政经济类。

所以我到底是个什么人？我在全网都叫“玩家翁伟”。人本来就是多面的，我只是把这种多面性稍微外露了一点。

## 技术的尽头，是好用的工具

2006年，我从新加坡国立大学计算机系毕业。

第一份工作，身边的同事不是博士就是硕士，全公司就我一个本科生。那家公司后来成了新加坡科技行业的“黄埔军校”，当年的同事们现在各个独当一面。当时的我，就是一个在里面拼命吸收养分的小卡拉米。

然后就是按部就班的一路打怪升级。从最基础的一线软件工程师，做到架构师，做到技术总监，再到CTO。我曾经帮一家跨境电商从A轮一路带到并购上市，也帮国内的企业在东南亚从零搭起区域研发中心。

真的是把人生最好的十几二十年，纯粹地砸在了技术里。

但人的认知是会变的。在我自己尝试了两次创业之后，我突然有了一种极其深刻的体会：**技术，真的只是手段而已。**

你代码写得再优雅，架构设计得再完美，如果做出来的产品没有社会价值，不能创造经济效益，那它就是一堆无用的字符。想通了这件事之后，我对技术的态度变得前所未有的明确——它就是工具。

我不再迷恋技术的自嗨，我只关心怎么用好工具，去创造真正的价值。

## 四十几岁，我迎来人生最高效的巅峰

然后，AI时代来了。

很多人在焦虑中年危机，焦虑程序员吃青春饭，但我现在四十几岁，我可以非常坦诚地说：**我现在正处在人生中学习速度最快、产出效率最高的巅峰期。** 这听起来有点反常识，但背后的逻辑很简单，因为我突然发现：**我以前做CTO的那些技能，在AI时代实现了完美的降维打击和全盘复用。**

以前我管理几百人的大型技术团队，前端、后端、运维、QA甚至产品线都要管。我的核心日常是什么？是跟团队讨论技术方案、定产品方向、搭技术架构，以及在团队遇到搞不定的技术难点时，亲自下场写关键代码。

现在呢？这些事情**一件都没变**。

唯一的区别在于，以前我需要跟真人同事开会、口头沟通、写文档交接；现在，我直接跟AI用文字讨论，敲定文档，然后让它去干活。以前我需要企业支付高昂的人力成本来维持一整个团队的运转；现在，我只需要付一笔微不足道的AI订阅费。

遇到AI写不出来的代码，我依然像当年那样，自己亲自动手解决核心问题。

**我依然在做技术管理，只不过我手下的兵，变成了一大群不知疲倦的AI。** 这是一种极其美妙的赋能，我一个人，就是一支军队。

## 在狂飙的AI时代，我选择了“手动化”

但在这个被AI按下了加速键的世界里，我却发现了一个反直觉的生存策略。

跟AI协作了快一年，我不仅没有变得越来越急躁，反而开始**主动放慢速度**。

现在的工具太方便了，很多人喜欢用语音输入，喜欢开发各种复杂的自动化Workflow来极限提速。我太理解这种冲动了，我年轻当程序员的时候，最喜欢干的就是自动化，觉得能让机器干的绝不自己动手。

但现在，我反而更喜欢“手动化”。我宁愿每次都慢慢地去手敲 Prompt，而不是追求那种极致的连贯和快捷。

为什么？**因为AI的生成速度太快了，快到如果我不强行降速，我的大脑会消化不过来。**

跟人类同事协作，效率的瓶颈在“等待”——你交代一个任务，可能要等半天或者隔天才能看到反馈；但跟AI协作，几分钟内它就能把一堆方案糊在你脸上。有时候我一不留神，它把这个也做了，那个也做了，但我自己可能都没想清楚：这个东西到底有什么用？我真的需要做这个吗？

所以我坚持打字。打字的过程，就是我强迫自己思考的过程。在敲下每一个字的时候，我可以斟酌措辞，可以修剪多余的想法，可以确保我对自己要解决的问题有着绝对清醒的认知。

## 学得够慢，很多东西就不用学了

这种刻意保持的“慢”，也让我避开了过去一年里AI圈子里的不少大坑。

前阵子行业里都在疯狂讨论 MCP（模型上下文协议），说这是颠覆性的技术。但我仔细看了一下，结合我对 AI Agent 工作原理的理解，我的判断很简单：这东西没啥意义。你只需要有标准的输入输出，能传输数据，大模型的能力就能发挥出来，搞那么复杂的抽象毫无必要。

所以我几乎没有在 MCP 上投入任何精力，没去写过，也没去试用过。

结果半年过去，行业的风向变了，开始有人出来反思和批评 MCP 的局限性了。今天早上我正好看到一个帖子，里面有一句话深得我心：**“在AI时代，只要你学得够慢，很多东西你就不用学了。”**

这并不是一种消极怠工，而是一种基于判断力的定力。

当你不再被眼花缭乱的新概念牵着鼻子走，当你学会了慢下来去思考事物的本质，你就会发现，这个时代真正稀缺的，从来不是掌握最新工具的速度。

**而是品味，是判断力，是你对真实世界问题的洞察力。** 工具会不断被淘汰，但这些沉淀在你脑子里的东西，才是你在这个时代立足的根本。
