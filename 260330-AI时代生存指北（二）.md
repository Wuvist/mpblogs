
# AI时代生存指北2：AI的死穴——能写商业计划书，端不稳一杯水

上一篇聊完，不少朋友在评论区打趣：翁老师，你这是要劝大家集体转行去当水管工吗？

其实，那句“去澳洲修水管”的建议，虽然听着像是个充满戏谑的段子，但它背后实则精准地击中了一个计算机科学界的经典命题。这个命题，或许就是我们普通人在AI时代最后的一道“护城河”。

# 聪明反被聪明误

早在1988年，机器人学家汉斯·莫拉维克就曾提出过一个让当时的AI界颇为尴尬的观察。他发现：**让计算机在智力测试或下国际象棋上达到成年人的水平，是相对容易的；但要让它拥有一个一岁婴儿那样的感知和行动能力，却难如登天。**

这就是著名的“**莫拉维克悖论**”。

如果用大白话来翻译，就是：人类觉得“难”的事情，比如复杂的逻辑推理、微积分运算、写长篇论文，对AI来说简直是小菜一碟；而人类觉得“简单”到甚至不需要动脑子的事情，比如走路不摔跤、从杂乱的抽屉里精准捏起一根针、或是察觉对面那个人说“没事”时其实很有事，AI至今依然做得一塌糊涂。

这种反差的原因其实深藏在进化的逻辑里。我们引以为傲的语言、逻辑和计算能力，在人类数亿年的进化史上不过是最近几千年才“打补丁”上去的新插件。它们高度结构化、规则明确，恰恰是计算机最擅长模拟的领域。

而那些被我们视作本能的移动、抓取和环境感知，是经过数亿年迭代、深深刻进神经系统的“隐性知识”。**你不需要思考就能接住一个飞来的苹果，但要写出一段程序让机械臂完成同样的动作，你会发现其中的参数复杂到令人绝望。**

所以，今天的AI可以几秒钟生成一份足以忽悠投资人的BP，但如果你让一个顶配机器人去乱糟糟的厨房里准确找到那瓶生抽，再稳稳端一杯水递给你，这依然是个世界级难题。

# 泥土里的护城河

莫拉维克悖论绝不仅仅是一个学术趣闻，它直接指出了一个极具现实意义的生存真相：**什么样的工作最难被替代？**

答案可能出乎很多精英的预料——不是那些端坐在写字楼里、靠处理符号和信息为生的知识型岗位，而是那些需要人的身体“真实在场”、在非标准化的物理环境中进行操作、并与活生生的人产生链接的工作。

一位水电工需要钻进每一栋结构迥异的房子，面对每一根走向成谜的管道，在狭窄潮湿的空间里做出实时判断。**没有两个施工现场是完全一样的，这种物理层面的随机应变，是目前任何数据化技术都无法覆盖的“黑盒”。**

同样，那些需要手指感受材料微妙反馈的手艺人，或是需要在面对面交流中传递安全感的心理咨询师、康复师，他们提供的价值里包含了“体温”和“在场感”。这种作为一个活人坐在另一个活人面前的确定性，屏幕那头的代码永远给不了。

当我们意识到这一点时，再看那个“修水管”的建议，你还会觉得是玩笑吗？当数字世界的脑力劳动被大规模压缩和稀释时，那些双脚踩在泥土里、双手与真实世界打交道的人，反而站在了最坚固的地面上。

# 身体本身就是一道护城河

我还有一位朋友，在那条朋友圈下面留了这么一段话：“**先躺平两年，不要急着跟风学那些所谓的AI工具，重点是锻炼好身体、调节好心态。慢慢来，反而比较快。**”

第一眼看去，你可能会觉得这人在消极避世。AI都快杀到家门口了，你竟然让我去健身？

但如果你真正理解了莫拉维克悖论，就会发现这其中蕴含着一种极深的生存智慧。AI在数字世界里的能力是指数级增长的，但在物理世界里的进展却异常缓慢。你的健康、体能和行动力，是你在这个世界上最基础的“硬件”。**AI没有身体，这是它最根本的死穴。而你，正拥有这道天然的屏障。**

当所有人都在拼命往数字能力的独木桥上挤时，回过头来投资自己的身体，听起来像是逆行，实则是在押注AI最薄弱的方向。

更何况，这场剧变不会在一夜之间落幕，它可能长达十年甚至更久。在这个过程中，你的身心状态决定了你能不能扛住那些突如其来的不确定性。**那些在焦虑中透支健康、试图通过“勤奋”来跑赢AI的人，往往还没等到转型的契机，就先把自己耗干了。**

# 别把孩子养成“大脑培养皿”

如果我们把视线从职场转回到家庭，这个逻辑就变得更加紧迫。

想想看，我们平时是如何安排孩子的时间的？上课、刷题、练琴、考证。每一分钟都极其“有用”，每一项活动都高度结构化。为了不让孩子“分心”，我们承包了所有家务；为了不“浪费时间”，我们压缩了他们的户外活动和社交。

**我们正倾尽全力把孩子培养成一个只长脑袋、不长身体的“大脑培养皿”。** 但扎心的是，这种在既定框架内进行的纯脑力输出，恰恰是AI火力最猛的射程正中心。

那些被你视为“浪费时间”的事，在AI时代反而成了最昂贵的战略投资。

让孩子去修理一个坏掉的玩具，去观察植物的生长，去菜市场跟摊主讨价还价，去面对一个没有标准答案的突发状况。**这些看似琐碎的经历，正在训练他们对物理世界的感知、对复杂人性的直觉，以及在混沌中寻找出路的本能。** 这些能力，比任何一门编程语言都更难被“编程”。

# 放大镜下的真相

写到这里，或许有人会产生一种错觉：既然物理世界这么稳，那我们都去干体力活好了。

当然不是。我并非在鼓吹每个人都去拧螺丝，我是在提醒大家，**如果你的思维和技能已经完全与真实的物理世界脱节，纯粹依赖规则性的逻辑吃饭，那你就正处于风暴的最前沿。**

而对于那些依然想在数字世界里耕耘的朋友，AI其实并非你的死对头。它更像是一面巨大的放大镜。你原来有多大本事，它就能帮你放大多少倍。

但这里有一个极其残酷的前提：如果你原本的本事是零呢？当一百万人都在用这面放大镜时，它放大的到底是什么？
