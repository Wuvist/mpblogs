
# 真相越追越远（中）：绝对正确的幻想

### 01 / 炒掉中间人，把神谕“Hardcode”下来

既然那班“肉身传话”的祭司信不过，人类顺理成章走出了下一步：**把 CEO 的旨意，白纸黑字全写下来。**

刻在石头上、抄在羊皮卷上。大家的底层逻辑是：只要文字写死不变，就相当于上了区块链，凡人没法随便篡改了，对吧？

于是，“经典”诞生了。永远正确、不可动摇、完美无瑕——在古代，这绝对是人类史上最硬核的信息技术。

真理公司的最高指示从“语音版”升级成了“文字版”，不仅能 **copy and paste（复制粘贴）**，还能一传十、十传百。底层打工人以后对业务有什么疑问，不用再去塞钱求中间人了，翻开《员工手册》，自己查 SOP。

犹太教在这方面一度把品控做到了极致。希伯来圣经的抄写员，连一个标点符号的间距都不敢擅自改动，生怕得罪天上那位至高无上的大老板。这套操作，看起来完美隔绝了人类的“主观污染”。

**但，真的是这样吗？**

### 02 / 醒醒，《员工手册》也是人编的

从考古学的角度看，这个文本系统从 V1.0 版本开始就自带漏洞。

20世纪中叶，有学者在以色列死海附近，挖到了目前已知最古老的圣经抄本——**死海古卷**。里面确实有大家熟悉的《创世记》等核心内容，但令人惊掉下巴的是：专家们同时在旁边发现了超过 20 份“外传”文本。

这就好比你买了个正版游戏，结果挖出了一大堆今天不存在的 **隐藏 DLC**。那些消失的文本里，写满了关于天使、魔鬼、人类族群的详细预言，甚至还有不少新角色。

当年住在那个洞穴附近的社群，显然认为自己手上的版本才是“官方正版”。可为什么这些内容今天全消失了？**是谁决定删掉它们的？**

这取决于你问谁。

如果你问神学家，他多半会坚信这本《员工手册》是天上直接掉下来的。一早装订好、排好版、带着目录，里面没有的内容，就是本来不该有的。

但如果你去问历史学家，他会无情地告诉你一个职场真相：**圣经其实是在不同历史阶段，由当时的拉比们（犹太教的高级学者/神职人员）坐在一起，像开千人大会一样疯狂辩论出来的。**

有些内容被支持，有些被反对；有些章节通过了审核绿灯，有些**就像选秀综艺一样被当场淘汰**。最终拍板定稿的根本不是神，而是由拉比们组成的“董事局”，通过举手投票决定的。

还记得吗？我们当初搞这套系统，初衷就是为了避开“人类天生会犯错”这个大坑。**结果最后负责给真理把关终审的那群人，全都是有立场、有主见、有私心的人类。**

学者 Harari 把这种状态叫做“绝对正确的幻想”（Illusion of infallibility）。过了几百年，死无对证，大家早就忘了当年那帮人是怎么开会砍需求的了。后人就把这本“精编版”当成了 100% 的神圣原文，一个字都碰不得。

### 03 / 致命 Bug：这是一本 Read-only 的手册

把古代文字奉为绝对真理，在实操中会遇到一个极其要命的问题：**它是 read-only（只读）的。**

世界变了，但规矩没有过期机制。

举个例子。犹太教有规矩说不能纹身。在古代，这条规矩其实很有道理。因为隔壁村的异教徒会用纹身来标识部族信仰，你不纹身，是为了跟竞品划清界限。这顶多算古代版的“品牌区隔”策略，算不上什么宇宙级的神圣法则。但几千年后的今天，这条规矩的原始商业语境早就连渣都不剩了。

再举一个更日常的黑色幽默。犹太教规定安息日“不能做工”。原意是为了保障打工人有休息日，初衷极好。但圣经里没给出 technical detail：**技术层面上，到底什么动作才叫“做工”？**

这就把后世的学者们搞疯了。

_手欠撕断一张纸巾，算不算做工？_

大家都知道，几千年前写给牧羊人看的规矩，作者怎么可能预判未来世界长什么样？

结果绝对不是开玩笑：现代的正统派犹太人，为了不违规，会在安息日开始前，提前把一天的纸巾全部撕好备用。后来人类发明了电，拉比们开会研究后宣布：按电灯开关等同于古代的“点火”，所以按开关也不行。

那如果你家恰好住二十楼怎么办？周末全天硬生生封印在家里？

于是，人类凭借无穷的智慧，发明了一样比周星驰电影里“太阳能手电筒”更精妙的东西：**安息日电梯（Sabbath elevator）**。

一到安息日，这部电梯就会自动停靠每一层。全程不需要你按任何键，你走进去像个木头人一样干站着就行了。**不按键 = 没做工。人类完美卡了上帝的 Bug，实现了“合法违规”。**

两千年后的今天，很多人依然死守着古代的金句，硬塞进现代生活。

经典从原本的生活指南，慢慢变成了拖住地球转动的锚。

### 04 / 无尽套娃，掉进“解读陷阱”

既然《员工手册》是只读的，没法出新版，那怎么处理跟现实脱节的部分呢？

绕了一大圈，这个任务又交回给了拉比们。

于是，公司里诞生了一个极具权力的体制内新 title：**经典解读专家（相当于神圣合规部）。**

拉比们将几百年的口水仗集结成书，专门教打工人怎么在“圣经已经过时”的年代里凑合过日子。但解读这东西，注定会引发更多的解读。后人又将这些神仙打架的对话，编成超级大部头，书里全是跨越几百、上千年的学者在互相引经据典地辩论。

**圣经里的每一句话，都变成了推论的起点。** 他们用新的 rules 去解释旧的 rules，用新的 law 去兜住旧的 law，疯狂打补丁。

举个经典例子。Torah（摩西五经）里有一句最出名的话：**“以眼还眼。”**

古代刚写下来的时候，原意就是字面意思：你弄伤我的眼，我就去弄瞎你的眼，讲究一个物理对等。

后来不知过了多少代，拉比们一拍大腿，重新解读说：_不对，真正的公义不能只靠报复，你弄瞎别人，并不能让受害者复明啊。_ 所以这句经文的“真正意思”其实是**金钱赔偿**——赔医药费和精神损失费。

你看懂这波操作了吗？

**他们改不了 CEO 的底层代码（原文），就直接去改前端 UI 的显示（意思）。**

当然，拉比们的对外口径很统一：我们从来没有曲解过，大老板原本就是这个意思。但事实摆在眼前：每个年代都有自己的一套解读，谁也不敢保证当前的最新版本以后会不会被推翻。

**重点来了：** 当初为什么要把神的话写成书？就是因为接受了“人类不可靠”，想炒掉所有中间人。

**但兜兜转转搞了一大圈，我们只是裁掉了一批中间人，又招了另一批中间人。**

各教各派的学者搞出了无数个不同的最终解释权，人类彻底掉进了一个套娃式的“解读陷阱”。

### 05 / 谁来垄断真理？

就在各说各话、系统快崩溃的时刻，有人发现：这样下去不行，队伍随时会散伙，必须进行一次彻底的**企业大重组（Restructuring）**。

这个强力的重组方案，是基督教阵营想出来的。他们发展出了一个史无前例的庞大机构——**罗马天主教会**——并一度拿下了对圣经的最终解释权。

罗马教廷将信仰高度中央集权化（Centralized）。那班主教和神学大师搞了好几次超级大会，把民间流传的各种野生经文做了一次大清洗。由**大台高层**直接拍板定下官方名录，把所有不合口味的野路子版本统统列为“禁书”。

为了维护官方版本的绝对权威，教廷还开了个新部门叫**宗教裁判所（Inquisition）**，专门做思想审查。

自此之后，**质疑公司，就等同于犯罪。**

历史上有一个令人毛骨悚然的真实案例。16世纪的意大利学者布鲁诺，不仅偷看禁书、讲“宇宙无限”，还敢公开否定教廷的神圣教条。结果他被关了整整七年，受尽酷刑都不肯收回言论。

行刑那天，据说教会为了封住他的嘴，用特制器具残忍地钉住了他的舌头。最后，这位只想追求宇宙真相的学者，就这样被活活烧死。

这就是挑战大台官方解读的下场。**当一个系统自认“绝对无错”的时候，连怀疑本身都是死罪。**

为了维持这种绝对控制，整个体系的学术环境开始严重内卷。当时那帮经学家每天的 KPI 就是：研究 A 神学家怎么解读 B 神学家对 C 神学家解读的解读。

整个欧洲最聪明的脑袋，全浪费在这种层层嵌套的“屎山代码”里。他们建起了一个密不透风的巨大回音室，跟外面的客观现实彻底脱节。

而这个系统最致命的 Bug 在于：**它彻底丧失了自我纠错机制（Rollback 机制）。**

正常人摸到滚烫的东西，自然会缩手。但如果一家巨型公司对外宣称，自己是由一个“绝对不会犯错的神”亲自掌舵的，它怎么发公关稿承认自己摸错了？

**只要一低头认错，整间公司的人设和神圣权威就瞬间塌了。**

唯一能保住公司的方法就是：永远死不认错，并解决提出问题的人。

**最终，秩序保住了。**

**但真相，死了。**

**👇 下期预告（大结局预警）：**

人类在“绝对正确”的泥潭里挣扎了上千年，直到一台机器的出现，彻底砸碎了大台的饭碗。

**下一篇，也是系列的最终章，我们聊聊：**

印刷机怎样打破了真理垄断？“绝对正确”这四个字，究竟杀死了多少无辜的人？

以及，科学革命真正的起点，为什么仅仅是因为人类敢于低头承认一句——**“我们其实什么都不知道”**？
