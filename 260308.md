很多不同的朋友来问我怎么“养龙虾”，甚至还有老友直接跟我说他想付费“听课”。

我很能理解朋友们，尤其是没有技术背景朋友们的这种焦虑。

是啊，现在是AI时代，不学AI的话，那会被时代淘汰。

怎么学？大家其实也不知道，但最近听说有个“龙虾OpenClaw”挺火的，我跟着学，那总没错吧？

我理解大家的焦虑，但我想说，大家不用焦虑，更不用去学什么“龙虾”。

千万千万不要被舆论的风潮给带偏了。

学AI是应该的，但其实不要想着通过“交费”来学习。

有很多人便是一冲动去买了mac mini，更烧了不少信用卡支付AI模型的token，去“养”龙虾。

但除非他们是最前沿的AI工作者，否则他们这样做，大概率是纯粹浪费了自己的时间跟金钱，到头来什么都没有学到。

龙虾只是一时的AI风潮，对于普通人来说，最好的方式就是“让子弹飞一会”，不要去当第一波“吃螃蟹/龙虾”的人，静观其变即可。

后面我仔细分析为什么普通人不应该吃龙虾；以及为何它会引起风潮，以及什么才是适合普通人的选择。

OpenClaw能够做的事情其实对于行内人士来说一点也都不稀奇。

什么自动“发推、写文”，甚至管理服务器、做交易分析等等等。

这些，其实AI早就能做了，并且行内人士，也很可能都在用了。

只是，我们使用的是AI开发工具，如Claude Code / gemini cli / codex等这样的命令行工具。

命令行工具是跑在黑咕隆咚的"命令行"里面的，但我日常做AI开发的工作，便是面对着这样的命令行，我们不用各种图形化界面，是因为命令行能够极大的提升我的工作效率以及精准的控制，打个其实不太恰当的比方说，赛车手，是一定要纯手动的，我们绝对不用“自动档”

我只要在命令行里跟AI说一句：“写一个程序，帮我抓取今天新加坡股市sgx上涨幅最高的十只股票，然后展示在网页报告中”

我的AI，便立刻会去分析如何抓取股票价格，这个API不行，就换另一个，甚至“自我反省”，总之一顿忙于十分钟，最后给我呈现了一个优雅的网页报告。

所以，在我们资深程序员眼里，龙虾能够做的事情，一点也不稀奇。

至于自动编辑、排版，多平台发文章等等，更是我十几年前就“玩剩下”的，我现在日常发文，早就不稀罕什么自动化工具，我就自己登陆各个平台，手动操作设定定时发布。我这样可以根据每篇文章给每个平台做微调，以确保可以更加符合相应平台的读者习惯，我觉得挺好的。

所以，AI能够做的事情早就非常多，非常“神奇”。（这里我要强调，人还是首先需要自己学会去做这些神奇的事，然后使用AI来放大自己的能力，加速自己的效率；绝对不要想着说使用AI来做自己不会做的事情，会“搬起石头砸自己脚，得不偿失”的）

但是，使用这些AI工具有“门槛”，首当其冲的便是需要使用那个黑咕隆咚的命令行（命令行其实对于AI本身来说，也是更加友好，所以我们会看到甚至谷歌最近也为自己的办公套件提供了命令行工具）。

对于普通人来说，命令行，恐怕就是跨不过去的门槛。

而OpenClaw，龙虾的伟大之处，便是把这样那样的门槛给移除了，让用户可以直接通过最为方便的IM聊天软件来跟AI对话，并让AI去做各种各样“神奇”的事情。

Tech科技的发展固然重要，人文关怀Liberal Arts更是重要；我还记得乔布斯在2011年iPad 2发布时的那段演讲。

平板电脑PC早就有了，但是使用有门槛，直到iPad的发布，让普通人，甚至小孩子，甚至猫猫狗狗都能使用的平板，这彻底解决使用科技的门槛，提升了科技的“可触及性”，这非常伟大。

从某种意义上说，龙虾也正是类似的伟大产物。

但它只是一个诞生了两三个月的产品，充满了“坑”，与其说它伟大，我更想说它疯狂。

把聊天软件跟AI工具打通这件事，其实一点也不难，我自己在两年前，为了方便国内的朋友使用Gemini，便做了个微信聊天机器人，直接把G宝拉到群聊里，AI能够看到群里信息，有“记忆”，而朋友想要问AI问题，直接在群里AT 相应的微信账号即可。

我能做，各家大厂当然也能做了。

不是想不到，而是大厂有责任感，他们不敢拿自己用户的数据与安全开玩笑。

“养龙虾”，Meta脸书负责安全的高管，都能养到把自己的邮箱给清空，然后她需要飞奔的去拔Mac mini的网线。

用马斯克的话来说，把自己数据、账号都交给龙虾，无异于拿一把AK47机关枪交给黑猩猩。

也许黑猩猩真的能拿着AK去帮你打猎，但它也完全可以随时掉转枪头。

所以，普通人绝对绝对不要去浪费自己的时间、金钱去“养龙虾”，毫无意义，并且危险。

还有人问我怎么使用龙虾去理财，我想说，那你就好好去学习理财知识，你自己懂得理财了，你才有可能利用AI去做更高效的理财。

不要心存侥幸，不要懒惰，不要想着有AI就可以让不懂理财的人掌握理财的技能，这就好比你不会打猎，然后想着让AI帮你打猎，然后你“养龙虾”，把机关枪给了一只黑猩猩龙虾，龙虾会跟你说，我已经瞄准好了，但你如果不会打猎，你是无法判断龙虾瞄准的，究竟是你的猎物，还是你自己。

现在有无数的创业团队，在打造各种AI产品，以求降低技术的门槛，提高可触及性。

海外的话，为了不让Claude Code被“局限在”命令行里，其背后的Anthropic公司已经推出Claude Coworks，要让非技术背景的白领，也能解锁AI的魔力，让他们更加高效的完成自己的专业工作。

国内，我相信字节、腾讯、阿里等企业，以及各种创业企业，也都会很快推类似的产品；作为普通人，我们等待企业去推相应的产品就好了。

而等待的同时，我们可以更加积极的去学习行业知识，去学习AI。

我会发现，很多普通人对于AI存在各种误解，以及不切实际的期望，真的，连怎么用好AI，如何跟AI对话都都搞不明白。

我会逐步讲述AI大语言模型是怎么工作，有什么长处跟短板，普通人应当如何利用好等等，谢谢关注。
