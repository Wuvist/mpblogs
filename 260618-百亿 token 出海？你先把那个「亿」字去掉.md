# token 是什么?

它是一个计量单位。跟「度」「字节」「公斤」是一类东西。

它本身不大。

可很多人一看到「十亿」「百亿」这种数字，被吓住了，以为是天大的体量。

不是的。

你记住一个换算：token 要除以一百万。

一百万个 token，撑死也就是几块钱，便宜的时候几分钱。

所以当有人敲锣打鼓喊「百亿 token 出海」的时候，你心里甚至可以默默把那个「亿」字划掉。

把它当成一百块钱的事来看。

就这么大。

真的，太小了，不值一提。
